matlab的使用

|总字数:4.2k|阅读时长:14分钟|浏览量:

MATLAB

基础

续行符

相当于C语言/n

注意使用方法

搜索路径

clear : 清除命令，删除所有变量

若出现变量名和函数名重名，变量名优先处理

先执行当前文件下的文件，之后才执行搜索路径下的文件。

注释

“%” 后面所有文字为注释.

数据类型

变量及其操作

（1）变量名必须是不含空格的单个词；
（2）变量名区分大小写；
（3）变量名最多不超过19个字符；
（4）变量名必须以字母打头，之后可以是
任意字母、数字或下划线，变量名中
不允许使用标点符号.

who 显示 变量名 whos显示全

内存变量文件

save:创建内存变量文件

load：装入内存变量文件

eg: save mydata a x

//把 a x 变量装在mydata.mat文件中

​ load mydata

// mydata 保存在当前工作的文件夹中。

数值数据

整形：无符号 8 16 32 64

​ 有符号8 16 32 64

uint8

将数值转化无符号八位整数

int8

数据转化成带符号八位整数

single()函数

其他类型转化成单精度

double()

其他类型数据转化成双精度

class()

查询变量类型

复型数据：

可用I或者j来表示虚部

real函数：求复数的实部

imag函数：求复数的虚部

format :

确定数据格式输出格式

​ eg: format long

函数

sum

exp函数:

以e为底的指数函数

三角函数：

数值若是弧度制则正常，若是角度，则在

​ 函数后加上d;

​ eg：sin(pi/2) == sin(90)

abs函数

对实数：绝对值 ；

复数：模长 ；

字符串 ASCII abs(‘a’)

取整函数

round : 四舍五入

ceil :向上取整

floor:向下取整

fix:舍弃小数

求余数函数

rem mod

isprime

判断一个数是否是素数

find

find()找到所有非零元素的位置

find(A==1)是1元素的位置

sqrt

开方函数

factorial(N)

计算阶乘

矩阵

建立矩阵：

基本方法

  1. 直接输入法：中括号括起来，同一行用空格或者逗号分开，不同行用分号 eg A=[1,2,3,4;5,6,4,7]

  2. 拼接矩阵：A一个矩阵，B一个矩阵

​ C=[A,B;B,A]

  1. 可以用实部矩阵和虚部矩阵构成复数矩阵

​ C=A+Bi

  1. 冒号表达式

e1:e2:e3 e1为初始值，e2为步长，e3为终止值

  1. linspace

linspace(a,b,n) a为第一个元素，b为最后一个元素，n为元素总数，当n省略时，自动产生100个元素

结构矩阵

格式为：结构矩阵元素.成员名 = 表达式

单元矩阵

建立单元矩阵和一般矩阵相似，直接输入就可以了，单元矩阵元素用大括号括起来，eg:b={10,’asd’,60}

空矩阵

a=[]产生一个空矩阵

zeros()——全零矩阵

b=zeros(m，n) 产生一个m行、n列的零矩阵

ones——全为1的矩阵

c=ones(m，n) 产生一个m行、n列的元素全为1的矩阵

eye——对角线、单位矩阵

d=eye(m，n) 产生一个m行、n列的单位矩阵

矩阵元素管理

赋值

eg:A(3.2) = 100, 如果要赋值的元素范围大于矩阵的范围，那么矩阵将扩充，并且空下来的位置赋值为0

引用元素

matlab中按照列存储，引用元素，可以用A（2，3）

也可以用序号

sub2ind函数：

将矩阵中指定元素的行列转换成存储的序号：D = sub2ind(S,I,J).S是行数和列数组成的下标，一般用size函数获取，I为行下标，J为列下标

计算矩阵中某一个元素在存贮中所处的位置

ind2sub函数：

序号转换成相应的下标

[I,J] = ind2sub(S,D)

矩阵元素操作

（1）矩阵A的第r行：A（r，：）

（2）矩阵A的第r列：A（：，r）

（3）依次提取矩阵A的每一列，将A拉伸为一个列向量：A（：）

（4）取矩阵A的第i1i2行、第j1j2列构成新矩阵:A(i1 : i2 , j1 : j2)

（5）以逆序提取矩阵A的第i1~i2行，构成新矩阵:A(i2 : -1：i1，：）

（6）以逆序提取矩阵A的第j1~j2列，构成新矩阵:A(:, j2 : -1：j1 ）

（7）删除A的第i1~i2行，构成新矩阵:A(i1:i2，：)=[ ]

​ 让要删除的元素变成空

​ eg:A(:,[2,4]) = []//让矩阵5和4列元素为空并且合并起来

（8）删除A的第j1~j2列，构成新矩阵:A(：， j1 : j2)=[ ]

（9）将矩阵A和B拼接成新矩阵：[A B]；

​ [A；B]

改成列向量

A（:）

函数

det(A)——方阵的行列式

inv（A）——方阵的逆

[V，D]=eig[A]——方阵的特征值与特征向量

求矩阵A的全部特征值，构成对角阵D，并求A的特征向量构成V的列向量。（V的每一列都是D中与之相同列的特征值的特征向量）

end运算符

表示某一维的末尾元素下标

reshape函数

reshape(A,m,n):在矩阵总元素保持不变的前提下将矩阵A重新排成m*n的二维矩阵 ，但不改变矩阵元素的存储序列。

size函数

返回的俩个整数，一个是矩阵的行数，第二个是矩阵的列数

repamat

B = repmat(A,m,n):将矩阵A复制m×n块，即把A作为B的元素，B由m×n个A平铺而成。

matlab基本运算

算术运算

+加 - * /右除 /左除 ~乘方

注：都是基于矩阵运算

点运算 .* ； ./ ；./；.~；

点运算相当于一个矩阵的元素进行运算 eg: A.*B == A和B中对应元素相乘

关系运算

< ;<=;>;>=;==;~=(不等于)

逻辑运算

& ， | , ~

inv求逆矩阵

字符串数据

单引号括起来的字符序列

若要表示一个单引号，则要用两个‘’

要建立多行字符串，形成一个字符串矩阵，那么对应列要相同

数字转化为字符串

num2str(c)

字符串合并函数

strcat(‘字符串1’,’字符串2’)

[‘字符串1’ ‘字符串2’]

eval

eval(S) ,S 是字符串，用matlab命令来执行S字符串

字符串和数值之间转换

abs和double 把字符串矩阵转换成ASCII数值矩阵

char把ASCII矩阵转换成字符串矩阵

字符串的比较

相同长度可以用关系运算符，以字符串ASCII值进行比对 eg’www0’>=’W123’ ans = 1 1 1 0

strcmp(s1,s2) : 比较两个字符串是否相同

strncmp(s1,s2,n): 比较两个字符串n个字符前是否相同

strcmpi(s1,s2): 忽略大小写比较两个字符串是否相同

strncmpi(s1,s2,n) : 忽略大小比较两个字符串n个字符前是否相同

字符串查找和替换

findstr(s1,s2):返回短的字符串在长字符串中的开始位置

strrep(s1,s2,s3)：将字符串s1中的所有子字符串s2替换成s3

数组数据

创建

  1. x=[a b c d e f ] 创建包含指定元素的行向量
  2. x=first：last
    创建从first开始，加1计数，到last结束的行向量
  3. x=first：increment：last
    创建从first开始，加increment计数，last结束的行向量
  4. x=linspace(first，last，n）
    创建从first开始，到last结束，有n个元素的行向量
  5. x=logspace(first，last，n）
    创建从开始，到结束，有n个元素的对数分隔行向量

数组元素访问

  1. 访问一个元素： x(i)表示访问数组x的第i个元素.
  2. 访问一块元素： x(a ：b ：c)表示访问数组x的从第a个元素开始，以步长为b到第c个元素（但不超过c），b可以为负数，b缺省时为1.
  3. 直接使用元素编址序号. x([a b c d]) 表示提取数组x的第a、b、c、d个元素构成一个新的数组
    [x(a) x(b) x(c) x(d)].

改成列向量

  1. 可以直接生成 eg: c=[1;2;3;4]
  2. 可以转置生成 eg: b=[1 2 3 4]; c=b’;

画图

曲线图

PLOT函数

PLOT(X,Y,S)，其中X,Y是向量,向量中两两对应，分别表示点集的横坐标和纵坐标，S表示线型，

eg: ·y 黄色 . 点 - 连线
·m 洋红 o 圈 : 短虚线
·c 蓝绿色 x x-符号 -. 长短线 r 红色 + 加号 – 长虚线

PLOT(X,Y)–画实线
PLOT(X,Y1,S1,X,Y2,S2,……,X,Yn,Sn) –将多条线画在一起

符号函数(显函数、隐函数和参数方程)

ezplot

ezplot(‘f(x)’,[a,b]) ——表示在a<x<b绘制显函数f=f(x)的函数图

ezplot(‘f(x,y)’,[xmin,xmax,ymin,ymax])——表示在区间xmin<x<xmax和 ymin<y<ymax绘制隐函数f(x,y)=0的函数图

ezplot(‘x(t)’,’y(t)’,[tmin,tmax])——表示在区间tmin<t<tmax绘制参数方程 x=x(t),y=y(t)的函数图

fplot

fplot(‘fun’,lims)——表示绘制字符串fun指定的函数在lims=[xmin,xmax]的图形.

注意：
[1] fun必须是M文件的函数名或是独立变量为x的字符串.
[2] fplot函数不能画参数方程和隐函数图形，但在一个图上可以画多个图形。

对数坐标图——坐标轴是10的多少次方

loglog(Y) 表示 x、y坐标都是对数坐标系
semilogx(Y) 表示 x坐标轴是对数坐标系
semilogy(…) 表示y坐标轴是对数坐标系
plotyy 有两个y坐标轴，一个在左边，一个在右边

空间曲线

一条曲线——PLOT3

PLOT3(x,y,z,s)——x,y,z，n维向量，分别表示曲线上点集的横坐标、纵坐标、函数值，s是指定颜色、线形等

多条曲线——PLOT3

PLOT3(x,y,z)

其中x，y，z是都是m*n矩阵，其对应的每一列表示一条曲线.

空间曲面

surf(x,y,z)

其中X,Y,Z，表示数据矩阵。分别表示数据点的横坐标、纵坐标、函数值。画出数据点（x，y，z）表示的曲面

Mesh(x,y,z)

其中X,Y,Z，数据矩阵。分别表示数据点的横坐标、纵坐标、函数值。画网格曲面，基于空间曲面的网格图

meshz(X,Y,Z)

在网格周围画一个curtain图(如,参考平面)

函数

subplot

subplot(a,b,c)//将这个界面分为a行b列第c个位置

grid on

grid on 在图上显示网格

rotate3d

rotate3d使得图片成3d图片

meshgrid

meshgrid(x,y)产生一个以x为行，Y为列的矩阵

图像处理光滑

shading faceted：默认模式，在曲面或图形对象上叠加黑色的网格线；

shading flat：是在shading faceted的基础上去掉图上的网格线；

shading interp：对曲面或图形对象的颜色着色进行色彩的插值处理，使色彩平滑过渡 ；

peaks

peaks:内部自带的山峰函数

pause

暂停函数，类似与C语言的pause

图像处理

注：所有操作都可以在图片上进行操作，**但操作完要保存**

在图形上加格栅、图例和标注——可在图片上的插入找到

grid ON: 加格栅在当前图上
grid OFF: 删除格栅

xlabel(string): 在当前图形的x轴上加图例string

ylabel(string): 在当前图形的y轴上加图例string

zlabel(string): 在当前图形的z轴上加图例string

title(string): 在当前图形的顶端上加图例string

gtext(‘string’)

命令gtext(‘string’)用鼠标放置标注在现有的图上.运行命令gtext(‘string’)时，屏幕上出现当前图形，在图形上出现一个交叉的十字，该十字随鼠标的移动移动，当按下鼠标左键时，该标注string放在当前十交叉的位置.

Axis 定制坐标 ——改坐标值

Axis([xmin xmax ymin ymax zmin zmax])x、y、z的最大、最小值，定制图形坐标

axis off 关闭坐标

图形保持——简化版就一个图上画多个线

hold on 保持当前图形, 以便继续画图到当前图上
hold of 释放当前图形窗口

打开一个新窗口

figure(h) 新建h窗口，激活图形使其可见，并把它置于其它图形之上

一般配套使用pause 使得在当前窗口下暂停，点击后继续出现另一个窗口

缩放图形

zoom on 为当前图形打开缩放模式

​ 单击鼠标左键，则在当前图形窗口中，以鼠标点中的点为中心的图形放大2倍；单击鼠标右键，则缩小2倍

zoom off 关闭缩放模式

改变视角——没用

（1）view(a,b)
命令view(a,b)改变视角到(a,b),a是方位角,b为仰角。缺省视角为（-37.5，30）。

（2）view（[x，y，z]）

​ view用空间矢量表示的，三个量只关心它们的比例，与数值的大小无关，x轴view（[1，0，0]），y轴view（[0，1，0]），z轴view（[0，0，1]）。

动画

Moviein(), getframe, movie()
函数Moviein()产生一个帧矩阵来存放动画中的帧；

函数getframe对当前的图象进行快照；

函数movie()按顺序回放各帧。

散点图: scatter（X,Y,S,C）

​ 在向量X和Y的指定位置显示彩色圈．X和Y必须大小相同．

平面等值线图：

contour (x,y,z,n) 绘制n个等值线的二维等值线图

空间等值线图：

contour 3(x,y,z,n) 其中n表示等值线数。

三维散点图

scatter3（X,Y,Z,S,C） 在向量X,Y和Z指定的位置上显示彩色圆圈. 向量X,Y和Z的大小必须相同.
